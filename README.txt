---------Atividade de Desenvolvimento WEB------------

Tela de Login:

Criado campos para preencimento com informações dos usuários, se o mesmo tiver cadastrado poderá acessar página principal
do sistema.

Caso o usuário não tenha cadastro poderá acessar a tela de registro de novo usuário.

Na tela de cadastro, o usuário poderá preencher os campos com as informações.

Feito o cadastro poderá acessar o sistema.

No menu principal haverá uma barra com 3 campos para navegar.

Abaixo vai conter todo o conteúdo do sistema.

Para vizualizar as informações, bastar clicar no nome do filme, série ou anime.

Dentro desse conteúdos haverá resumo de filme, série ou anime.

Observação: nem todos os filmes, animes e séries serão possíveis acessar, pois ainda não foi disponibilizados nenhum conteúdo.
